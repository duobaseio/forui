# Forui Widget Spotlight

An internal playground app the Forui team uses to showcase [Forui](../forui) widgets, primarily to record clips for
the [Widget Spotlight series](https://x.com/foruidev).

> [!IMPORTANT]
> This is **not** a getting-started demo or sample app. If you are looking to try out Forui in your own project, see 
> the [documentation](https://forui.dev/docs) instead.
