Forui 提供遵循 [llms.txt](https://llmstxt.org/) 约定的机器可读文档端点。这些端点便于 LLM 与 AI 驱动的工具理解并使用 Forui。

## 端点

### `/docs/llms.txt`

一份简明索引，汇总所有文档页面的标题、描述与链接。可作为查阅可用文档的起点。

[查看 llms.txt](/docs/llms.txt)

### `/docs/llms-full.txt`

以纯 Markdown 形式呈现的完整文档内容，包含所有页面的代码示例、API 详情与使用说明。

[查看 llms-full.txt](/docs/llms-full.txt)
