Forui 与 [Flutter Hooks](https://pub.dev/packages/flutter_hooks) 提供一流的集成。所有控制器都在配套的 `forui_hooks` 包中以 hook 的形式暴露。

## 安装

在你的 Flutter 项目目录下运行以下命令安装 `flutter_hooks` 与 `forui_hooks`。

```bash filename="bash" copy
flutter pub add flutter_hooks
flutter pub add forui_hooks
```
