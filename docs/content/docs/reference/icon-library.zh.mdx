Forui Assets 是一个 Flutter 库，提供一组来自 [Lucide](https://lucide.dev/) 的高质量图标。

## 安装

<Callout type="info">
    Forui Assets 已包含在 forui 包中。如果你已安装 `forui`，无需再单独安装。
</Callout>

在你的 Flutter 项目目录下运行以下命令安装 Forui Assets。

```bash filename="bash" copy
flutter pub add forui_assets
```

## 用法

<Callout type="info">
    查找图标列表的最佳方式是访问 [Lucide](https://lucide.dev/icons/) 网站。
    我们会随每个 Forui 版本一同更新 Forui Assets 中的图标。
    如果你发现 Forui Assets 中缺少某个图标，欢迎[提交 issue](https://github.com/duobaseio/forui/issues/new)。
</Callout>
