Es gibt **2** Wege, einzelne Widget-Styles anzupassen: **Deltas** für kleine Anpassungen und das **CLI** für größere Umbauten.

## Deltas

Zu jedem Widget-Style gehört ein passendes Delta, das eine Teilmodifikation beschreibt. So besitzt
[`FAccordionStyle`](https://pub.dev/documentation/forui/latest/forui.widgets.accordion/FAccordionStyle-class.html) etwa ein
zugehöriges [`FAccordionStyleDelta`](https://pub.dev/documentation/forui/latest/forui.widgets.accordion/FAccordionStyleDelta-class.html).

Widgets nehmen Deltas über ihren `style`-Parameter entgegen. Um die Fokus-Outline-Farbe eines Accordions zu ändern,

### Varianten

Widget-Styles enthalten [`FVariants`](/docs/concepts/themes#variants)-Felder. Sie lassen sich über eine
`delta(...)`-Funktion anpassen, die eine Folge von Operationen entgegennimmt. Je nach Typ der Variante sind diese
Operationen entweder Delta-basierte Modifikationen oder vollständige Ersetzungen.

Es gibt **3** Wege, `FVariants`-Felder anzupassen:

Zusammengesetzte Typen wie `BoxDecoration`, `IconThemeData`, `TextStyle` und verschachtelte Styles unterstützen
Delta-basierte Modifikationen. Um den Titel-Textstil eines Accordions zu ändern:

## CLI

Der folgende Abschnitt zeigt, wie der Style eines Accordions überschrieben wird. Dieser Weg empfiehlt sich für größere
Style-Umbauten.

Erzeugen Sie einen Widget-Style mit:

```shell copy
dart run forui style create accordion
```

<Callout type="info">
    Tipp: Mit `dart run forui style ls` sehen Sie alle verfügbaren Styles.
</Callout>
