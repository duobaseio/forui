## Verschachtelter Header

Er wird typischerweise auf verschachtelten Seiten oder Sheets verwendet.

<Tabs items={['Vorschau', 'Code']}>
  <Tab>
    <Widget name='header' variant='nested'/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={nestedSnippet} />
  </Tab>
</Tabs>

## CLI

Um einen bestimmten Style zur Anpassung zu erzeugen:

<Tabs items={['Headers', 'Header Action']}>
    <Tab>
        ```shell copy
        dart run forui style create headers
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create header-action
        ````
    </Tab>
</Tabs>
