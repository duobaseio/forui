### Behavior

#### Swipe Navigation

The tab content area can be scrolled horizontally when `expands` is true.

- **Defaults:** True on mobile (touch devices) and false on desktop. Use `swipeablePhysics` to override.
- **Physics:** Defaults to `BouncingScrollPhysics`.
