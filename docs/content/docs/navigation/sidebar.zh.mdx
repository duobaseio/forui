## 结构

本节描述 sidebar 各个元素的结构。

### `FSidebar`

为屏幕侧边提供风格统一布局的 widget。该 widget 包含以下区块：

| 名称    | 参数           | 位置       | 是否可选 |
| :------ | :------------- | :--------- | :------- |
| header  | header         | 固定       | 是       |
| content | child/children | 可滚动     | 否       |
| footer  | footer         | 固定       | 是       |

### `FSidebarGroup`

用于将多个 `FSidebarItem` 组合在一起的 widget。

### `FSidebarItem`

表示 sidebar 上的一个项。可以嵌套在 `FSidebarItem.children` 中以创建嵌套项。
