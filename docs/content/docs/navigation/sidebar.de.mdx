## Aufbau

Dieser Abschnitt beschreibt den Aufbau der verschiedenen Elemente einer Sidebar.

### `FSidebar`

Das Widget, das ein vorgefertigtes Layout am seitlichen Bildschirmrand bereitstellt. Das Widget besitzt die folgenden Bereiche:

| Name    | Parameter      | Position    | Optional |
| :------ | :------------- | :---------- | :------- |
| header  | header         | fest        | Ja       |
| content | child/children | scrollbar   | Nein     |
| footer  | footer         | fest        | Ja       |

### `FSidebarGroup`

Ein Widget, das verwendet wird, um mehrere `FSidebarItem`s zu gruppieren.

### `FSidebarItem`

Stellt einen Eintrag in der Sidebar dar. Kann in `FSidebarItem.children` verschachtelt werden, um verschachtelte Einträge zu erzeugen.

### Sheet-Sidebar

Geeignet für Geräte mit begrenztem Bildschirmplatz.
