## CLI

Um einen bestimmten Style zur Anpassung zu erzeugen:

<Tabs items={['Item Group', 'Items', 'Item', 'Item Content']}>
    <Tab>
        ```shell copy
        dart run forui style create item-group
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create items
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create item
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create item-content
        ````
    </Tab>
</Tabs>

#### Mehrere Gruppen zusammenführen

Diese Funktion führt mehrere `FItemGroupMixin` zu einer einzigen Gruppe zusammen. Sie eignet sich, um eine Gruppe mit mehreren Abschnitten darzustellen.

<Tabs items={['Vorschau', 'Code']}>
  <Tab>
    <Widget name='item-group' variant='merge' height={300}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={mergeSnippet} />
  </Tab>
</Tabs>
