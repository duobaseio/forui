## CLI

要生成可自定义的特定样式：

<Tabs items={['Item Group', 'Items', 'Item', 'Item Content']}>
    <Tab>
        ```shell copy
        dart run forui style create item-group
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create items
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create item
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create item-content
        ````
    </Tab>
</Tabs>

## 示例

### 行为

#### 可滚动

<Tabs items={['预览', '代码']}>
  <Tab>
    <Widget name='item-group' variant='scrollable' height={200}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={scrollableSnippet} />
  </Tab>
</Tabs>

#### 懒加载滚动

<Tabs items={['预览', '代码']}>
  <Tab>
    <Widget name='item-group' variant='lazy' height={300}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={lazySnippet} />
  </Tab>
</Tabs>

#### 合并多个分组

该函数将多个 `FItemGroupMixin` 合并为一个分组，便于呈现包含多个区段的分组。

<Tabs items={['预览', '代码']}>
  <Tab>
    <Widget name='item-group' variant='merge' height={300}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={mergeSnippet} />
  </Tab>
</Tabs>
