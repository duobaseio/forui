</Callout>

## CLI

Um einen bestimmten Style zur Anpassung zu erzeugen:

<Tabs items={['Items', 'Item', 'Item Content']}>
    <Tab>
        ```shell copy
        dart run forui style create items
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create item
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create item-content
        ````
    </Tab>
</Tabs>
