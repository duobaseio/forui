日历在移动端支持滑动手势，用户可以左右滑动以在不同页面之间切换。

[`FCalendarController`](https://pub.dev/documentation/forui/latest/forui.widgets.calendar/FCalendarController-class.html)
控制日期选择行为，包括单选日期、多选日期和日期范围选择。

<Callout type="info">
    `FCalendar` 与所有 `FCalendarController` 返回的 `DateTime` 均为 UTC 时区，并截断到最接近的整天。
</Callout>

<Callout type="info">
  在桌面端和 Web 上滚动时按住 `Shift` 可在日期之间切换。
</Callout>
