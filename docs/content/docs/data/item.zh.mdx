## CLI

要生成可自定义的特定样式：

<Tabs items={['Items', 'Item', 'Item Content']}>
    <Tab>
        ```shell copy

        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create item
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create item-content
        ````
    </Tab>
</Tabs>
