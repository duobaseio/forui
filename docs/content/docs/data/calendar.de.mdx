Der Kalender unterstützt Wischgesten auf Mobilgeräten, sodass Nutzer durch Wischen nach links oder rechts zwischen Seiten navigieren können.

Ein [`FCalendarController`](https://pub.dev/documentation/forui/latest/forui.widgets.calendar/FCalendarController-class.html)
steuert das Auswahlverhalten für Daten, einschließlich Einzeldatum, mehrerer Daten und Datumsbereich.

<Callout type="info">
    `FCalendar` und alle `FCalendarController` geben `DateTime`-Werte in der UTC-Zeitzone zurück, abgeschnitten auf den nächsten Tag.
</Callout>

<Callout type="info">
  Halten Sie auf Desktop und Web beim Scrollen `Shift` gedrückt, um durch die Daten zu navigieren.
</Callout>
