## Beispiele

### Gruppe

Umschließen Sie mehrere Tooltips mit einem `FTooltipGroup`, damit nachfolgende Tooltips sofort erscheinen, sobald das erste angezeigt wurde.

<Tabs items={['Vorschau', 'Code']}>
  <Tab>
    <Widget name='tooltip' variant='group' height={300}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={groupSnippet} />
  </Tab>
</Tabs>

### Horizontale Ausrichtung

Sie können ändern, wie der Tooltip am Button ausgerichtet wird.

<Tabs items={['Vorschau', 'Code']}>
  <Tab>
    <Widget name='tooltip' variant='horizontal' height={300}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={horizontalSnippet} />
  </Tab>
</Tabs>

### Nur Langes Drücken

<Tabs items={['Vorschau', 'Code']}>
  <Tab>
    <Widget name='tooltip' variant='long-press-only' height={300}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={longPressOnlySnippet} />
  </Tab>
</Tabs>
