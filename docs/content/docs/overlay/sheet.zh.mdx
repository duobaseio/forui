它每次都会导航到一个新页面。

<Callout type="info">
    与之密切相关的 widget 是 [持久化 sheet](/docs/overlay/persistent-sheet)，它显示作为应用主要内容补充的信息，
    且不会阻止用户与应用交互。
</Callout>

## CLI

要生成可定制的特定样式：

<Tabs items={['Modal Sheet']}>
    <Tab>
        ```shell copy
        dart run forui style create modal-sheet
        ````
    </Tab>
</Tabs>
