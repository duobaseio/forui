## CLI

To generate and customize this style:

```shell copy
dart run forui style create tooltip
```

## Usage

### `FTooltip(...)`
<UsageSnippet usage={tooltipUsage} />

## Examples

### Group

Wrap multiple tooltips in an `FTooltipGroup` so that subsequent tooltips appear instantly after the first one is shown.

<Tabs items={['Preview', 'Code']}>
  <Tab value="Preview">
    <Widget name='tooltip' variant='group' height={300}/>
  </Tab>
  <Tab value="Code">
    <CodeSnippet snippet={groupSnippet} />
  </Tab>
</Tabs>

### Horizontal Alignment

You can change how the tooltip is aligned to the button.

<Tabs items={['Preview', 'Code']}>
  <Tab value="Preview">
    <Widget name='tooltip' variant='horizontal' height={300}/>
  </Tab>
  <Tab value="Code">
    <CodeSnippet snippet={horizontalSnippet} />
  </Tab>
</Tabs>
