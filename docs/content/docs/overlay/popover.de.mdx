## Beispiele

### Verschachtelte Popovers

Wenn Sie Widgets, die intern Popovers verwenden (z. B. `FSelect`), innerhalb eines `FPopover` platzieren, schließt sich
das äußere Popover bei der Interaktion mit dem Dropdown des inneren Widgets. Das liegt daran, dass das innere Dropdown
in einer eigenen Overlay-Ebene gerendert wird und ein Tippen darauf als "außerhalb" des äußeren Popovers gilt.

Um dies zu verhindern, geben Sie beiden Widgets dieselbe `groupId`.

<Tabs items={['Vorschau', 'Code']}>
  <Tab>
    <Widget name='popover' variant='nested' height={500}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={nestedSnippet} />
  </Tab>
</Tabs>

### Horizontale Ausrichtung

Sie können ändern, wie das Popover am Button ausgerichtet wird.

<Tabs items={['Vorschau', 'Code']}>
  <Tab>
    <Widget name='popover' variant='horizontal' height={500}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={horizontalSnippet} />
  </Tab>
</Tabs>

### Klick außerhalb schließt das Popover nicht

<Tabs items={['Vorschau', 'Code']}>
  <Tab>
    <Widget name='popover' variant='no-hide-region' height={500}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={noHideRegionSnippet} />
  </Tab>

### Spiegelung entlang der Achse

Das Popover kann entlang der überlaufenden Achse gespiegelt werden, um innerhalb der Viewport-Grenzen zu bleiben.

<Tabs items={['Vorschau', 'Code']}>
  <Tab>
    <Widget name='popover' variant='flip' height={500}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={flipSnippet} />
  </Tab>
</Tabs>

### Verschiebung entlang der Achse

Das Popover kann entlang der überlaufenden Achse verschoben werden, um innerhalb der Viewport-Grenzen zu bleiben.

<Tabs items={['Vorschau', 'Code']}>
  <Tab>
    <Widget name='popover' variant='slide' height={500}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={slideSnippet} />
  </Tab>
</Tabs>

### Überlauf zulassen

Das Popover wird auch dann nicht verschoben, um innerhalb der Viewport-Grenzen zu bleiben, wenn es überläuft.

<Tabs items={['Vorschau', 'Code']}>
  <Tab>
    <Widget name='popover' variant='allow-overflow' height={500}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={allowOverflowSnippet} />
  </Tab>
</Tabs>
