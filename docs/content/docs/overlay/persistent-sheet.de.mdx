Es ist Bestandteil von [FScaffold](/docs/layout/scaffold), das in den meisten Fällen bevorzugt werden sollte.

<Callout type="info">
    Ein eng verwandtes Widget ist das [modale Sheet](/docs/overlay/sheet), das die Nutzer daran hindert, mit dem
    Rest der App zu interagieren.
</Callout>

<Callout type="warn">
    Alle Aufrufe von `showFPersistentSheet(...)` sollten innerhalb von Widgets erfolgen, die entweder `FScaffold` oder
    `FSheets` als Vorfahren haben.
</Callout>
