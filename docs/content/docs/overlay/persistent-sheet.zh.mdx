它是 [FScaffold](/docs/layout/scaffold) 的一部分，多数情况下应优先使用 FScaffold。

<Callout type="info">
    与之密切相关的 widget 是 [模态 sheet](/docs/overlay/sheet)，它会阻止用户与应用的其余部分交互。
</Callout>

<Callout type="warn">
    所有对 `showFPersistentSheet(...)` 的调用都应在拥有 `FScaffold` 或 `FSheets` 作为祖先的 widget 内部进行。
</Callout>
