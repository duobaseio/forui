<Callout type="info">
    Verwenden Sie zum Anzeigen eines Toasts die Funktionen `showFToast(...)` oder `showRawFToast(...)`. Die Funktion
    muss aus einem Nachfahren von `FToaster` aufgerufen werden.

    Wir empfehlen, `FToaster` in der `builder`-Methode von `MaterialApp`/`WidgetsApp`/`CupertinoApp` zu platzieren:
    ```dart
    MaterialApp(
      builder: (context, child) => FTheme(
        data: FThemes.neutral.light,
        child: FToaster(child: child!),
      ),
      home: HomePage(),
    );
    ```
</Callout>
