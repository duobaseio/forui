<Callout type="info">
    要显示 toast，请使用 `showFToast(...)` 或 `showRawFToast(...)` 函数。该函数必须从 `FToaster` 的后代中调用。

    我们建议将 `FToaster` 放在 `MaterialApp`/`WidgetsApp`/`CupertinoApp` 的 `builder` 方法中：
    ```dart
    MaterialApp(
      builder: (context, child) => FTheme(
        data: FThemes.neutral.light,
        child: FToaster(child: child!),
      ),
      home: HomePage(),
    );
    ```
</Callout>
