Es navigiert jedes Mal zu einer neuen Seite.

<Callout type="info">
    Ein eng verwandtes Widget ist das [persistente Sheet](/docs/overlay/persistent-sheet), das ergänzende Informationen
    zum Hauptinhalt der App anzeigt, ohne die Nutzer an der Interaktion mit der App zu hindern.
</Callout>

<Tabs items={['Vorschau', 'Code']}>
  <Tab>
    <Widget name='modal-sheet' height={500}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={defaultSnippet} />
  </Tab>
</Tabs>

## CLI

Um einen bestimmten Style zur Anpassung zu erzeugen:

<Tabs items={['Modal Sheet']}>
    <Tab>
        ```shell copy
        dart run forui style create modal-sheet
        ````
    </Tab>
</Tabs>
