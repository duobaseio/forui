## 用法

### `FTappable(...)`
<UsageSnippet usage={tappableUsage} />

### `FTappable.static(...)`

`FTappable` 的一个不带任何动画的变体。其效果类似于使用 `FTappableMotion.none`。

<UsageSnippet usage={tappableStaticUsage} />

## 自定义弹跳动画

你可以通过传入 `FTappableMotion` 来自定义 tappable 的弹跳动画。

<Tabs items={['预览', '代码']}>
  <Tab>
    <Widget name='tappable' variant='bounce' height={300}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={tappableBounce} />
  </Tab>
</Tabs>

## 滑动切换交互

可以将多个 tappable 放入一个 `FTappableGroup` 中以启用滑动切换交互。当你按下其中一个 tappable 并滑动到另一个上时，按下状态会转移到新的
tappable。

  </Tab>

  </Tab>
</Tabs>
