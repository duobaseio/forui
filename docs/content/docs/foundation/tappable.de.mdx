## Slide-across-Interaktion

Mehrere Tappables lassen sich in einer `FTappableGroup` gruppieren, um eine Slide-across-Interaktion zu aktivieren.
Wenn Sie ein Tappable drücken und auf ein anderes hinüberziehen, wird der gedrückte Zustand auf das neue Tappable
übertragen.
