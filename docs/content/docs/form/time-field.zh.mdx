<Callout type="info">
    建议在触摸设备上使用 [FTimeField.picker](#ftimefieldpicker)，在非主要触摸设备上使用 [FTimeField.new](#ftimefield)。
</Callout>

输入字段同时支持方向键导航：

- 上/下箭头：递增/递减值
- 左/右箭头：在时间片段之间移动

输入字段不支持以下使用非西文数字以及需要组合的文字的语言区域，会默认使用英语：

- 阿拉伯语
- 阿萨姆语
- 孟加拉语
- 波斯语/法尔西语
- 马拉地语
- 缅甸语
- 尼泊尔语
- 普什图语
- 泰米尔语
- 阿姆哈拉语
- 卡纳达语
- 韩语
- 旁遮普语
- 泰语

以下语言区域将默认使用中文 (zh)：

- 中文（香港）
- 中文（台湾）
