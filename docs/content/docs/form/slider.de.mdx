#### Markierungen

Markierungen bieten visuelle Bezugspunkte auf der Slider-Schiene. Jede Markierung besteht aus:

- Einem Strich: ein visueller Indikator, der die Position der Markierung anzeigt
- Einer Beschriftung: Text, der den Wert der Markierung beschreibt (optional)

<Tabs items={['Vorschau', 'Code']}>
