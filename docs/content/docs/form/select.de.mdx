## CLI

Um einen bestimmten Style zur Anpassung zu erzeugen:

<Tabs items={['Select', 'Select Search', 'Select Content', 'Select Section', 'Select Scroll Handle']}>
    <Tab>
        ```shell copy
        dart run forui style create select
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create select-search
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create select-content
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create select-section
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create select-scroll-handle
        ````
    </Tab>
</Tabs>
