---
title: Picker
description: Ein generischer Picker zur Auswahl eines Eintrags. Er besteht aus einem oder mehreren Rädern, optional mit Trennern zwischen den Rädern. Der Picker unterstützt die Navigation per Pfeiltasten. Empfohlen für Touch-Geräte.
apiReference: https://pub.dev/documentation/forui/latest/forui.widgets.picker/
---

Der Picker unterstützt die Navigation per Pfeiltasten:

- Pfeiltasten Auf/Ab: Ausgewählten Wert erhöhen/verringern
- Pfeiltasten Links/Rechts: Zwischen den Rädern wechseln

Empfohlen für Touch-Geräte.
