## CLI

要生成可定制的特定样式：

<Tabs items={['Text Field Sizes', 'Text Field']}>
    <Tab>
        ```shell copy
        dart run forui style create text-field-sizes
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create text-field
        ````
    </Tab>
</Tabs>
