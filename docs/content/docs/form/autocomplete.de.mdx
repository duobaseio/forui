description: Ein Autocomplete liefert eine Liste von Vorschlägen basierend auf der Eingabe des Nutzers und zeigt einen Typeahead-Text für den ersten Treffer an. Es ist ein Form-Feld und kann daher in einer Form verwendet werden.

<Callout type="info" title="Kein durchsuchbares Select">
    Ein Autocomplete ist kein durchsuchbares Select. Es ist ein Textfeld mit Vorschlägen. Werte sind nicht auf einen
    der Vorschläge beschränkt; Nutzer können Beliebiges eingeben. Wenn Sie ein durchsuchbares Select benötigen,
    verwenden Sie [ein durchsuchbares Select](/docs/form/select#searchable)
    oder [ein durchsuchbares Multi-Select](/docs/form/multi-select#searchable)
</Callout>

## CLI

Um einen bestimmten Style zur Anpassung zu erzeugen:

<Tabs items={['Autocomplete', 'Autocomplete Field Sizes', 'Autocomplete Field', 'Autocomplete Content', 'Autocomplete Section']}>
    <Tab>
        ```shell copy
        dart run forui style create autocomplete
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create autocomplete-field-sizes
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create autocomplete-field
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create autocomplete-content
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create autocomplete-section
        ````
    </Tab>
</Tabs>
