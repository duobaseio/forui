Der Picker unterstützt die Navigation per Pfeiltasten:

- Pfeiltasten Auf/Ab: Ausgewählten Wert erhöhen/verringern
- Pfeiltasten Links/Rechts: Zwischen den Rädern wechseln

Empfohlen für Touch-Geräte.

<Tabs items={['Vorschau', 'Code']}>
    <Tab>
        <Widget name='time-picker'/>
    </Tab>
    <Tab>
        <CodeSnippet snippet={defaultSnippet} />
    </Tab>
</Tabs>

## CLI

Um einen bestimmten Style zur Anpassung zu erzeugen:

<Tabs items={['Time Picker']}>
    <Tab>
        ```shell copy
        dart run forui style create time-picker
        ````
    </Tab>
</Tabs>
