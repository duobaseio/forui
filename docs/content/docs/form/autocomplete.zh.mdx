title: 自动补全
description: autocomplete 根据用户输入提供建议列表，并为首个匹配项显示前导提示文本。它是一个表单字段，因此可在表单中使用。
apiReference: https://pub.dev/documentation/forui/latest/forui.widgets.autocomplete/
---

<Callout type="info" title="不是可搜索的 select">
    autocomplete 不是可搜索的 select，它是一个带建议的文本字段。值不限于建议中的某一项，用户可以输入任何内容。如果你需要可搜索的 select，请使用
    [可搜索的 select](/docs/form/select#searchable)
    或 [可搜索的 multi select](/docs/form/multi-select#searchable)
</Callout>

## CLI

要生成可定制的特定样式：

<Tabs items={['Autocomplete', 'Autocomplete Field Sizes', 'Autocomplete Field', 'Autocomplete Content', 'Autocomplete Section']}>
    <Tab>
        ```shell copy
        dart run forui style create autocomplete
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create autocomplete-field-sizes
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create autocomplete-field
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create autocomplete-content
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create autocomplete-section
        ````
    </Tab>
</Tabs>
