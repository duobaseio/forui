<Callout type="info">
    建议在触摸设备上使用 [FDateField.calendar](#fdatefieldcalendar)，在非主要触摸设备上使用
    [FDateField.new](#fdatefield)/[FDateField.input](#fdatefieldinput)。
</Callout>

输入字段同时支持方向键导航：

- 上/下箭头：递增/递减值
- 左/右箭头：在日期片段之间移动

输入字段不支持以下使用非西文数字的语言区域，会默认使用英语：

- 阿拉伯语
- 阿萨姆语
- 孟加拉语
- 波斯语/法尔西语
- 马拉地语
- 缅甸语
- 尼泊尔语
- 普什图语
- 泰米尔语
