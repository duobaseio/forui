`FDialog` does not provide a built-in content layout. Generate one with the CLI or copy it from an example below, then
customize the widget directly.

## CLI

To generate a specific style for customization:

<Tabs items={['Dialog', 'Dialog Route']}>
    <Tab value="Dialog">
        ```shell copy
        dart run forui style create dialog
        ````
    </Tab>
    <Tab value="Dialog Route">
        ```shell copy
        dart run forui style create dialog-route
        ````
    </Tab>
</Tabs>
