## CLI

Um einen bestimmten Style zur Anpassung zu erzeugen:

<Tabs items={['Tile Group', 'Tiles', 'Tile', 'Tile Content']}>
    <Tab>
        ```shell copy
        dart run forui style create tile-group
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create tiles
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create tile
        ````
    </Tab>
    <Tab>
        ```shell copy
        dart run forui style create tile-content
        ````
    </Tab>
</Tabs>

#### Mehrere Gruppen zusammenführen

Diese Funktion führt mehrere `FTileGroupMixin` zu einer einzigen Gruppe zusammen. Sie ist nützlich, um eine Gruppe mit
mehreren Abschnitten darzustellen.

<Tabs items={['Vorschau', 'Code']}>
  <Tab>
    <Widget name='tile-group' variant='merge' height={300}/>
  </Tab>
  <Tab>
    <CodeSnippet snippet={tileGroupMerge} />
  </Tab>
</Tabs>
